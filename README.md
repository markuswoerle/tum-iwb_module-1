# Research Data
This repository contains the research data used in the manuscript "Technical suitability assessment of manufacturing system components for hydrogen-based high-temperature processes" by Markus Woerle, Svenja Schrader, Jan Geier, Daniel Schneider, Christina Reuter, and Michael F. Zaeh (accepted, Procedia CIRP, DOI: tbd.).

## Contact
Markus Woerle (markus.woerle@iwb.tum.de)

## Documentation
To replicate the results shown in the paper mentioned above, all the files that are stored in this repository are to be downloaded and to be saved in the same folder.

### MATLAB files
The following files contain the MATLAB code that was used to evaluate the case study in the paper:
- main.m
- susceptibility.m
- requirements.m
